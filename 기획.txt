마우스 포인트 이동 :
 해당 위치를 마우스로 클릭 -> 마우스 커서의 위치를 가져온다. -> 가져온 위치로 캐릭터를 이동시킨다.
 -> 행동종료

기본 공격 : 무기에 따라서 달라야함.
 해당 위치를 마우스로 클릭 -> 마우스 커서의 레이캐스트로 액터의 위치를 가지고 온다.
 -> 공격이 들어가기 전까지 액터의 위치를 계속해서 가지고 온다. 
 -> 무기의 최대 사거리 안쪽에 캐릭터가 도달하면 공격 모션 실행, 이 때 몬스터의 위치는 그만 가져온다.
 -> 행동 종료
